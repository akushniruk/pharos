# Pharos V1 Roadmap

Date: 2026-04-02
Status: Approved

Back to [Docs Portal](../../README.md).

## Vision

Pharos is a desktop app that observes AI coding agents (Claude, Codex, Copilot) running on your machine — zero setup, no hooks, no per-project config. Install, run, see everything.

## Phases

### Phase 1: Cleanup
Remove dead code (Go server, Python hooks), update docs, make daemon-rs the single backend.

### Phase 2: Cross-platform + Tauri Desktop App
Desktop-first. Single binary via Tauri. Linux/Windows path support.

### Phase 3: UI Redesign — Project Dashboard
Vercel/Railway-style project-centric dashboard replacing flat event list.

### Phase 4: V1 Polish
Final pass before sharing with real users.
